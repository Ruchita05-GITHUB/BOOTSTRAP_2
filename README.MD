# Bootstrap 5 UI Exploration

## Description
A responsive Bootstrap 5 web application developed using Handlebars and Vite as part of the internship assignment.

## Features
- Responsive Navbar
- Hero Section
- About Page
- Contact Page
- Bootstrap Cards
- Contact Form
- Responsive Layout

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Handlebars
- Vite

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
